# Город как текст: топонимика Новосибирска

Сайт итогового проекта по курсу «Цифровые технологии в профессиональной деятельности». Сделано в [Quarto](https://quarto.org/), опубликовано через GitHub Pages.

## Структура

- `index.qmd` — исходник главной страницы
- `_quarto.yml` — конфигурация сайта
- `images/` — картинки
- `toponym_map.html` — интерактивная карта
- `docs/` — собранный сайт

## Локальная сборка

```bash
quarto render      # 
quarto preview     
```
